## Purpose
Hub for the Hormozi playbook *$100M Playbook: Lifetime Value — Eight Ways to Get Customers to Buy More Stuff*. Holds distilled framework notes derived from this playbook. Lifetime Value is a tactical playbook centered on one operating thesis: **he who can make a customer more valuable than his competition wins** — and on a single core framework (the Crazy Eight) that decomposes "make customers spend more" into eight discrete operational levers any operator can run against any product or service.

This hub does not contain raw book content. Provenance is carried via `source_reference` frontmatter on each canonical note plus the Source Map.

## Reuse vs. New Decisions

This playbook reuses concepts already canonicalized in the vault. Canonical home stays where it is; this book maps in via the Source Map and links out from the Crazy Eight note. Reuses:

- **LTV / LTGP calculation** (gross profit, churn, transaction count, both formulas) — defined in `04-100M-Lost-Chapters/`. The Lifetime Value PDF explicitly states this section is "excerpts from the lost chapters given away from my book launch." Same math, same examples. See [[../04-100M-Lost-Chapters/Hormozi-CFA-Three-Levers|CFA Three Levers]].
- **Pricing posture / commodity trap / virtuous cycle** — defined in `01-100M-Offers/`. The Crazy Eight specifies *how* to test price; the strategic *why* stays there. See [[../01-100M-Offers/Hormozi-Pricing-Power|Pricing Power]].
- **Upsell mechanism architecture** (Classic / Menu / Anchor / Rollover) — defined in `03-100M-Money-Models/`. Crazy Eight Levers 5–6 specify *which dimension to lift* (quantity / quality); the *how to present* stays there. See [[../03-100M-Money-Models/Hormozi-Upsell-Offers|Upsell Offers]].
- **Downsell mechanism architecture** (Payment Plan / Trial-with-Penalty / Feature Downsell) — defined in `03-100M-Money-Models/`. Crazy Eight Levers 7–8 specify the qualified-only rule and visitor-revenue math; mechanism architecture stays there. See [[../03-100M-Money-Models/Hormozi-Downsell-Offers|Downsell Offers]].
- **Recurring offer architecture** — defined in `03-100M-Money-Models/`. Lever 3's "add recurring" sub-mechanic links there. See [[../03-100M-Money-Models/Hormozi-Continuity-Offers|Continuity Offers]].
- **90-day reactivation cadence + quarterly promotion mechanic** — defined in `06-Fast-Cash/`. Lever 3's "follow-up" sub-mechanic and the "every 90 days" promo logic live there. See [[../06-Fast-Cash/Hormozi-Fast-Cash-Cadence|Fast Cash Cadence]] and [[../06-Fast-Cash/Hormozi-Fast-Cash-Play|Fast Cash Play]].

This book contributes three things not previously in the vault: the Crazy Eight umbrella itself; the nine-tactic cost-reduction palette; and the operational Conv × LTGP price-testing procedure.

## Canonical Home Rule
The three frameworks above are canonical in this folder — they are specific to LTV-lifting and not duplicated elsewhere. LTGP math (Lost Chapters), pricing posture (100M Offers), upsell/downsell mechanism architecture (Money Models), continuity (Money Models), and quarterly cadence (Fast Cash) remain canonical in their respective folders; this book links to them rather than re-canonicalizing.
