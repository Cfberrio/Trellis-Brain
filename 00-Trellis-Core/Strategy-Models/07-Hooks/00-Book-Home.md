## Purpose
Hub for the Hormozi playbook *$100M Playbook: Hooks — How to Double Your Marketing in Under 5 Seconds*. Holds distilled framework notes derived from this playbook. Hooks is a short, tactical playbook with one operating thesis: **the hook decides whether the rest of the asset gets consumed at all** — and a small, repeatable system (two-part anatomy, format palette, 70-20-10 generation process, swipe library) for producing hooks that double or 10× attention without changing any other variable in the funnel.

This hub does not contain raw book content. Provenance is carried via `source_reference` frontmatter on each canonical note plus the Source Map.
