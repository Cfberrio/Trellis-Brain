# Price Raise — Book Home

## Purpose
Hub for the Hormozi playbook *$100M Playbook: Price Raise — How to Raise Prices Without Losing Customers*. Holds distilled framework notes derived from this playbook. Price Raise is a short, tactical playbook centered on one operating thesis: **the easiest way to make more money is to charge more for the same thing**, executed via a disciplined rollout (test on new customers → segment old customers → send the RAISE letter → handle the three response types) protected by a fixed set of operating rules and a quantitative test that proves the raise is profitable before it touches the existing base.

This hub does not contain raw book content. Provenance is carried via `source_reference` frontmatter on each canonical note plus the Source Map.

## Reuse vs. New Decisions (vs. 01-100M-Offers)
This playbook restates the virtuous-cycle / premium-pricing posture from `$100M Offers` and applies it operationally to **existing customers** (not new offers). The cycle and the Dan Kennedy "no strategic advantage to second-cheapest" quote remain canonical in `01-100M-Offers/Hormozi-Pricing-Power`. Price Raise links there rather than re-canonicalizing.

- **Pricing posture, virtuous cycle, race-to-the-bottom** — defined in [[../01-100M-Offers/Hormozi-Pricing-Power|Pricing Power]]. Price Raise extends with operational variables (emotional investment, demandingness, conviction in sales process, gratitude) inside [[Hormozi-Price-Raise-Play|Price-Raise-Play]] but does not re-canonicalize the principle.
- **Value Equation logic** behind "raise value to justify the raise" — defined in [[../01-100M-Offers/Hormozi-Value-Equation|Value Equation]]. Price Raise references it inside the RAISE letter's Invest section.
- **Scarcity / Urgency** as offer levers — defined in `01-100M-Offers/`. Price Raise uses an *expiring loyalty discount* as a softening lever (a vanishing-discount mechanic), not as a new framework. Documented inside [[Hormozi-RAISE-Letter|RAISE-Letter]].
