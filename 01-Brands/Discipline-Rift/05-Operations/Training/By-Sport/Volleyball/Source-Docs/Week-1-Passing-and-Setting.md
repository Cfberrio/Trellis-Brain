### Page 3

This needs to be crystal clear for new coaches, so hereʼs the exact version:
Setup
All players start on the center line, spread out (no pushing).
Coach stands where everyone can see/hear.
Court overview (60–90 sec max)
Walk to each line and point:
End line / baseline
Sidelines
Center line
Attack line 10-foot line)
Optional later: "net line" / "under the net" is not a line—donʼt teach that as a
line.)
Instructions (practice rounds first)
Coach says: "When I call a line, you sprint to it and touch it with your hand.
Then freeze in athletic stance."
Coach calls 3 practice rounds:
"Attack line!" →  they run/touch/freeze
"Sideline!" →  run/touch/freeze
"End line!" →  run/touch/freeze
Game version (choose ONE)
Version 1 Elimination (original)
Last player to touch the line goes to Job Zone (not "out forever").
After 10 seconds, they re-enter next round.
Version 2 Team points (faster, fewer stoppages)
Split into 2 teams.

B) SHUFFLING CRAB GAME (posture + platform + movement)
This is your "3 Pʼs" game — keep it exact.
Teach "Crab Position" with the 3 Pʼs (30 sec)
Posture: low, knees bent, chest tall
Platform: hands together, arms straight (ready to pass)
Position: feet wider than shoulders, weight on toes
Setup
Players spread out facing the coach (enough space to shuffle without
collisions).
Coach holds a volleyball as the visual cue.
Instructions
Everyone starts in Crab Position.
Coach moves the ball:
right / left / forward / backward
Players must shuffle the direction the ball moves while staying low.
Challenge rule (keeps it fun)
Coach may say "right" but move the ball left.
Players must follow the ball, not the voice.

Block 6 — Application (Constraint Game) (12–18 min)
GAME OF THE DAY: JAILBREAK (clarified + paced)
Setup
Split into Group A and Group B on opposite sides.
Each group lines up behind the baseline.
Coach has balls at the net or attack line for quick feeds.
How it runs (fast cycle)
First athlete steps to attack line.
Coach feeds:
early rounds = toss
later rounds = faster toss / light serve
Athlete must pass OR set (coach calls which, or athlete decides if you want
DR.
Ball must go over and in.
Lives + jail
Everyone starts with 3 lives.
Miss (into net/out/not over) = lose a life.
At 0 lives →  go to jail (other side).
Breaking out (keep it safe + skill-based)
Instead of only "catch a ball," make it skill aligned:
Catch = allowed only if coach calls it safe
OR breakout option:
"2 clean platform catches" (self-toss →  platform pop →  catch) in jail area
then re-enter
