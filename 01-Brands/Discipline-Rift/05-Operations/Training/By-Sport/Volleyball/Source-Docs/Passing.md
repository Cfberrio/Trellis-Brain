PASSING

PASSING MODULE
Developmental Volleyball)
Purpose of This Module
This module defines how we teach passing to beginners.
The focus is not perfect passes.
The focus is building correct habits through:
sequence
movement
simple, shared cues
Every coach teaches passing the same way, using the same language, so players
build habits faster and with less confusion.

### Page 2

Teaching Philosophy
Beginner players:
Cannot process long technical explanations
Learn best through imagery + repetition
Need one correction at a time
Because of this, we teach passing through:
Numbered sequences
Short verbal cues
Consistent language across all coaches
Cues are corrections, not explanations.
Core Concept: Passing Is a Sequence
Passing is not one action.
It is a chain of actions, done in order.
We teach this order using numbers so players:
Move before passing
Use their legs
Stay balanced
PASSING BY NUMBERS (FOUNDATION)
Position 1 — Ready
Cue: "Ready"
Athletic stance
Knees bent
Back straight
Arms out and relaxed

### Page 3

Ready to move
No platform yet
Important clarification:
Arms stay relaxed until the player has moved into the ballʼs path.
Why:
Players must react first.
Building a platform too early locks the body and reduces balance and adjustment.
Position 2 — Platform
Cue: "Pancakes"
Hands together (hand over hand)
Thumbs together = "Syrup"

### Page 4

Forearms form one surface
Additional cue: "Surfboards"
Arms stay flat and firm
No bending at the elbows
Why:
A bent platform creates unpredictable passes.
A flat, quiet platform allows the ball to rebound cleanly.
Position 3 — Low
Cue: "Low"
Knees bend

### Page 5

Hips drop
Chest stays tall
Important clarification:
Hips go down — not chest forward.
Why:
Power comes from the legs, not collapsing posture.
Staying tall preserves balance, vision, and platform control.
Position 4 — Up
Cue: "Up"
Legs extend

### Page 6

Finish tall
Why:
The ball goes up because the body goes up.
Arms never add power.
They only provide direction.
Core Passing Pattern
1 →  2 →  3 →  4
Ready →  Platform →  Low →  Up
This sequence is non-negotiable.

### Page 7

Key Teaching Cues (Coach Logic)
"Pancakes & Syrup"
Used when:
Hands separate
Thumbs are not together
Platform is uneven
Creates:
Wide, stable platform
Fast visual correction
"Surfboards"
Used when:
Elbows bend
Arms swing
Platform collapses
A surfboard is:
Long
Flat
Never bent
This cue reminds players:
The platform must stay straight and firm through contact.
HOW PASSING PROGRESSES OVER TIME
The technique never changes.
Only the movement before the pass changes.

### Page 8

Early Phase (Week 1 Focus)
Goal: Shape and confidence
Teaching emphasis:
Ready position
Correct platform shape
Leg usage
How coaches should teach:
Mostly stationary or very light movement
Slow, controlled reps
Stop drills only if the same error appears repeatedly across players
Primary cues:
"Ready"
"Pancakes"
"Surfboards"
"Low →  Up"
Application Phase (Around Week 4)
Goal: Passing while moving and defending space
Movement patterns are added before the pass.
The passing technique stays the same.
Pattern A — Step & Pass
Used when the ball is slightly off center.
Sequence:
Ready
Step
Platform

### Page 10

Coaching Priorities to Remember
Movement comes before platform
Legs create the pass
Arms stay flat and quiet
Cues are corrections, not explanations
Same language across all coaches
Coach Reminder
If a pass is off, ask yourself:
Did they move first?
Was the platform flat?
Did they use their legs?
Fix one thing at a time.
