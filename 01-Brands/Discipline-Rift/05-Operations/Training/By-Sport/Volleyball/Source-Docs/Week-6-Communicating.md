Block 5 — DR Teaching (Guided Decisions)
Learning Target: Choose pass vs set based on ball height
Cue:Above forehead = set. Below chest = pass.
Activity:READ & REACT SHUFFLE
How it works
low toss →  shuffle and pass
high toss →  shuffle and set
vary only one thing at a time: left/right/short/deep
Why this belongs here
Block 5 is the first place full See →  Decide →  Act belongs. Your activity does that
correctly because players must read height, choose the contact, and then send
the ball to control. The source base supports the distinction:
overhead passing/contact occurs at face or forehead level / above shoulder
height
forearm passing is used when players move feet to ball and create a forearm
platform
So your above/below rule is a good developmental simplification for this week.
