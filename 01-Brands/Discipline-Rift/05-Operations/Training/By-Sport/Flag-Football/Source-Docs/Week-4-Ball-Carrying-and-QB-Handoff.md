### Page 2

"Today weʼre learning how to start fast after the whistle: QB hands it, RB runs it—
secure. Our cue is tuck + lock it away. Effort norm: reset fast so everyone gets
reps."
Micro-CFU (10s)
Choral response: "Whatʼs the cue?" →  "Tuck + lock it away!"
Quick scan: who answers loud + ready stance.
Block 2 — Retrieval Warm-Up (7 min)
Learning Target
Retrieve last weekʼs YAC habits: catch →  run hard →  protect the ball.
Activity: Music-Stop YAC Chase (your B2)
Pairs with a ball. Light toss/catch while music plays.
When music stops + whistle: ball carrier tucks + locks and sprints to a cone
line; partner chases 1520s.
Beginner version: straight run, big space.
Experienced version: must "show lock" on the first 2 steps (coach is watching).
Micro-CFU (10s "Freeze")
Whistle →  everyone freezes mid-carry. Coach scans: 4 points of contact (hand,
forearm, bicep, chest).
One fast fix cue only: "Lock it away."
Block 3 — Teaching Block (8 min)
Learning Target
Install the handoff start: whistle →  handoff →  tuck/lock →  run downhill.

### Page 3

QB job (no extra cue—just job description):
"Two hands, place it in the belly" (firm, not tossed)
RB job (tie back to cue):
Receive →  tuck + lock it away immediately →  eyes up →  run straight
Blocked Practice (6 min)
Groups of 3 in a lane: QB  RB + "shadow" defender (walk pace, no pulls).
Rep starts only on whistle. QB hands off. RB runs 810 yards.
Micro-CFU (15s)
"Show me the cue on step 2."
Exit criterion to move on: 8/10 reps = clean exchange + secure carry by step 2.
Block 4 — Skill Acquisition Reps Engine (12
min)
Learning Target
Get volume + confidence on handoff →  secure run.
Activity: Handoff Snake Run Lanes (blocked →  serial)
Set 46 snake lanes with cones.
Tier System (same drill, different demand):
Tier 1 Blocked): whistle →  handoff →  run the snake, no defender.
Tier 3 Serial whistle →  handoff →  run snake →  finish through a "gate" cone.
Tier 5 Constraint): whistle →  handoff →  run snake while keeping ball outside
arm when turning (advanced only).
Rotation: 3 reps RB then swap QB ↔ RB. Keep lines short 23 kids per lane max).

Block 5 — Teaching Acquisition (Guided

Learning Target
See →  Decide →  Act under light pressure: protect the ball while a defender

Activity: Handoff + Chase Read
Same lanes, now add a defender behind RB.
Progression:
Shadow (walk): defender follows—RB must keep secure carry.
Chase (game pace): defender tries to pull flag after 2 steps.
Advanced read (only if theyʼre ready):
"If defender is closing on your ball side, switch to outside armwithout losing
tuck/lock."
Coach teaching moves (fast)
Perception Q "What did you notice about where the defender was?"
Replay immediately: "Now choose the outside arm and lock it."
Micro-CFU (10s)

### Page 5

Learning Target
Transfer to a game: whistle start + handoff + YAC finish with secure carry.
Constraint Game: YAC Points (4v4) — handoff required
Rule: Every play begins on whistle with a QB → RB handoff (no passes this game).
Field: short width, 1020 yards to end zone.
Scoring:
1 point if runner crosses goal line
1 "Security Bonus" if runner maintains tuck + lock the whole run (coach
decides fast)
Coach focus: praise the process: "You locked it away through contact—point!"
Micro-CFU between rounds (10s)
"See →  Decide →  Act in one sentence."
Example: "I saw the defender close left, decided to switch outside, and ran."
Gamefy (Week 4)
Ball Security Shield
Random whistle during game: everyone freezes and shows tuck + lock in
under 2 seconds.
Team earns the shield when every player nails it.
