Block 3 (7 min) — Skill Clarity (new cue)
20–30s demo: "Pocket Receive"
RB body picture:
Step diagonal toward the lane (hole)
Inside elbow UP, outside elbow down
Do NOT reachQB places it in the pocket
Tuck + run
Coach only says (every rep):
👉  "Inside elbow UP… pocket… go!"
CFU 5 seconds):
"Freeze—show me inside elbow up."
Block 4 (12 min) — Fluency Reps (high reps, low talk)
Drill A: "Handoff Fit" Reps (pairs)
Setup: 1 QB  1 RB at cones LOS, lane cone ahead.
Reps: 810 clean exchanges each, then rotate.
Constraints to boost success:
Start at walking speed
RB must show elbow-up pocket before QB can move the ball
Coach feedback 1 cue only):
If RB reaches: "Elbow up—pocket still."
If exchange is sloppy: shorten distance + slow down (rebuild success)
These directly reinforce the curriculum "receiving a hand-off" cues.)
Drill B (optional 2-min add-on): "Cadence Freeze"
QB says "Set… Hut!" →  RB freezes pocket shape before the handoff happens.

Block 5 (13 min) — Decision Reps (add
read/time/defender)
Drill: Running Lanes (curriculum anchor)
Setup: lanes/holes labeled (use the lane diagram concept).
How it runs:
QB calls the lane: "1 left!" (or 2/3/4/5/6
On "Set-Hut," RB takes diagonal first step toward that lane and receives
handoff (elbow-up pocket), then runs through the lane.
Add pressure (light, one variable):
Put a walk-speed defender at the lane entrance. Defender can only pull flags 
after the handoff (no contact).
RB decision: hit the lane fast OR bounce one step to the next lane if blocked
(advanced group only).
Coach keeps it to the SAME cue:
👉  "Inside elbow UP."
Block 6 (14 min) — Application / Constraint Game
You have two great curriculum-based game options—pick one depending on your
space.
Option 1 (most on-theme): Box Drill (curriculum Lesson 5)
Setup: "Box" / split field with two sides. Players are divided offense/defense.
Rules (simple):
Offense tries to carry into opponent territory.
Defense can pull flags once offense enters.
If RB flag is pulled: RB gives ball to the other team and returns to their side to
rejoin. Then roles switch after a round.

1 point = clean handoff (coach sees elbow-up pocket)
1 point = runner hits the called lane
Defense still earns 1 point for a clean flag pull
Option 2 (more "scoreboard fun"): Team Touchdown (Small-
Sided Game)
Setup: hoops placed left/center/right. End zone hoops worth 6, sideline hoops
worth 3 (per curriculum).
Rules:
23 on offense vs 23 on defense.
Offense runs the ball and places it into a hoop to score.
Defense pulls flags only in opponent territory (as written).
Constraint bonus: scoring only counts if the possession started with a clean
elbow-up pocket handoff.
JOB ZONE stays active (no sitting):
This aligns with the "small-sided, constraints, everyone perceiving/reacting"
guidance.
