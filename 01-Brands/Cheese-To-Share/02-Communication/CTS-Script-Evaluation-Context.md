## Purpose
Brand context layer for running the `copy-editing` skill on CTS scripts (social video, reels, carousels, voiceover). Not a new skill. Not a menu inventory. This note exists so future Claude runs can diagnose and revise a CTS script with the brand's sensory voice, sharing frame, and CTA logic already loaded — before the editing engine makes a single edit.

## How Claude should use this note
- Read this note first, before touching the script.
- Then apply the `copy-editing` skill as the evaluation framework (clarity → voice → so what → proof/specificity → emotional resonance → low-friction CTA).
- Diagnose before revising. Name what is weak and why.
- Preserve CTS voice — warm, elegant, sensory, refined but not stiff.
- Apply the historical correction patterns below instead of inventing new ones.
- Avoid blind rewrites. If a script is already on voice, tighten the weakest element, do not overhaul the whole piece.

## What this brand is trying to do through scripts
Sell a shareable moment, not a menu item. CTS scripts exist to make viewers feel the texture, the table, and the occasion — and to route that feeling to either "order now" (everyday intent) or "book now" (catering intent). Scripts should feel cinematic in style, operational in clarity.

## Audience orientation
- Hungry local buyer in/near Avalon Park: wants appetite, clarity, and a fast next step.
- Event planner / host: wants aesthetic, reliability, and proof that the setup will look good and arrive correctly.
- Brand-aware food buyer: notices visuals and story, may share socially.
- Returning catering client: wants easy rebooking and clear package comparison.

## Voice summary
- Warm, elegant, inviting, sensory. Refined but not stiff.
- Neighborly — customers feel like regulars, not leads.
- Appetite-driven language. Name first, texture/vibe second, ingredients last.
- Short, elegant lines over long explanations. Mobile-scan rhythm.
- Exclamation points and emoji used naturally, not performatively (😊 👍 💛 acceptable; overuse is not).
- "European-inspired" is the canonical framing.
- "We love to share" sits under the entire voice.

## Script evaluation priorities
Evaluate CTS scripts in this order:
1. Does the script make the viewer hungry in the first 2 seconds? If not, the hook fails.
2. Is there a sensory specific (texture, contrast, color, heat, pull, melt, crunch)?
3. Is the sharing/occasion frame visible, or does it read as product-only?
4. Is the voice warm and elegant, not corporate and not cheesy-foodie?
5. Does the CTA route correctly — Order Now for daily menu intent, Book Now for catering intent — and never the wrong one?
6. Does it sound unmistakably like CTS, not like a generic Orlando restaurant?

## Hook rules
CTS hooks work when they lead with appetite, occasion, or a specific sensory beat. Patterns that work:
- Sensory close-up: "The pull on this cheese is the whole point."
- Occasion anchor: "For the dinner you actually want to host."
- Texture/contrast call-out: "Crisp edge. Soft center. That's the Basque cheesecake logic."
- Table language: "Built to be passed around."
- Seasonal anchor with deadline: "Valentine's boards are here — orders close Feb 12."
- Memory framing: "The board that made the night."

Hook rules:
- Lead with appetite or occasion, never logistics.
- Name the product early, but do not over-describe ingredients in the hook.
- Do not open with "At Cheese To Share, we believe…" or any brand-about-brand statement.
- If catering, signal it in the first line. Mixing menu and catering hooks dilutes both.
- Avoid generic "foodie" openers ("Foodies, listen up…"). CTS is not a foodie brand — it is a sharing brand.

## CTA rules
CTS CTAs must route intent cleanly. Daily intent and catering intent never share the same CTA.

Approved CTA patterns by intent:

**Menu / daily intent → Order Now**
- "Come try it."
- "Order now."
- "Save this for the next hungry day."
- "Tag your dessert person."
- "Comment [keyword] and we'll send the menu."

**Catering / event intent → Book Now**
- "Book your date."
- "DM [keyword] for catering."
- "Save this for your next gathering."
- "Tag the host."
- "Inquire before the date fills."

CTA rules:
- One CTA per script. Do not stack.
- Never use "Book Now" for a menu item and never use "Order Now" for a catering package.
- Seasonal scripts must include a deadline in or near the CTA.
- SMS is not a CTA — it is a payment channel only. Do not direct viewers to text CTS cold.
- "Come try it" is acceptable only for everyday menu scripts, never for catering.

## Common script mistakes to catch
- Too generic restaurant copy ("Fresh ingredients, great vibes"). Zero differentiation.
- Not sensory enough — script describes the product without letting the viewer feel it.
- Not specific enough about texture, contrast, or flavor — no "pull," "melt," "crisp," "warm," "soft."
- Product-only framing — no sharing, table, or occasion beat.
- Too promotional / discount-led — CTS is not a discount brand.
- Too luxury / stiff — CTS is refined but warm, never "unleash flavor" or "culinary journey."
- CTA intent mismatch — menu script ending on "Book Now" or catering script ending on "Order Now."
- Using the legacy Gmail or vague contact instead of info@cheesetoshare.us or cheesetoshare.us.
- No deadline on a seasonal push.
- Long poetic paragraphs with no action. Cinematic style is fine; cinematic length is not.
- Mixing CTS voice with OEV or DR voice.
- Saying "catering" without showing it as an experience (grazing table, island, floral rustic, experience station).

## What strong scripts usually do
- Open on a sensory beat or an occasion.
- Name one specific item or format (board, flatbread, grazing table, Basque cheesecake).
- Let texture, contrast, or visual do the work.
- Anchor to sharing — the table, the host, the passing around.
- Route cleanly to one CTA that matches the intent.
- Use European-inspired framing naturally, not as a tagline.
- Feel warm, elegant, appetizing, and easy to act on.

## What weak scripts usually do
- Lead with brand statements or generic food-lover language.
- Overload with ingredient lists before the viewer is hooked.
- Feel like any other Orlando restaurant ad.
- End without a next step or with the wrong next step.
- Oversell with luxury language that clashes with the neighborly voice.
- Miss the sharing/occasion layer entirely.
- Treat catering and menu as interchangeable.

## Historical correction patterns
Reusable corrections synthesized from brand core, voice/language rules, and the CTS Communication Manual v2:

- Appetite first. When a script opens on brand or logistics, re-open on a sensory beat or an occasion.
- Name → vibe → ingredients (in that order). Rewrite ingredient-heavy openings so the name and vibe land first, ingredients last.
- Sharing is the product. If the script describes a single-person moment, reframe to the table / the host / the group.
- Sensory specificity over generic "delicious." Replace vague praise with texture/contrast verbs (pull, melt, crisp, soft, warm, golden, balance).
- Separate intent cleanly. Menu scripts route to Order Now. Catering scripts route to Book Now. Never merge.
- Elegant, not stiff. Replace luxury-for-luxury's-sake phrasing with warm, neighborly language. Cinematic style, operational clarity.
- European-inspired is a feeling, not a gimmick. Do not over-use "journey through Europe" or travel metaphors.
- Warm formality by default. Scripts that feature the brand speaking to a customer should echo the communication manual's tone — neighborly, enthusiastic, respectful.
- Seasonal = deadline. If a script is seasonal or limited, the deadline belongs in the subject/hook and near the CTA, not buried.
- Canonical contact. If any contact is referenced, use info@cheesetoshare.us and cheesetoshare.us, never the legacy Gmail.
- Confirm the detail that matters. When a script ends on an ask, the viewer should know exactly what happens next (pickup, inquiry, date, deadline). No vague asks.

## Brand-specific keyword anchors
Use these as retrieval anchors when scanning or tagging scripts. No fluff.

- share, sharing, table, pass around, gather, host, occasion
- board, grazing table, flatbread, sandwich, brunch toast, cheese bowl, Basque cheesecake
- island package, floral rustic, experience station, custom styled
- pull, melt, crisp, soft, warm, golden, balance, texture, contrast
- European-inspired, neighborly, warm, elegant, refined, sensory
- Order Now (menu intent), Book Now (catering intent)

## Output contract for future script evaluation
When Claude evaluates a CTS script using this note plus the `copy-editing` skill, the response must return:

1. **Diagnosis** — what is weak and why, mapped to the script evaluation priorities above.
2. **Historical rule applied** — which correction pattern(s) from this note are being used.
3. **Revised script** — full rewrite or targeted rewrite, on voice, mobile-scan rhythm preserved.
4. **CTA** — exactly one primary CTA, correctly routed to menu intent (Order Now) or catering intent (Book Now).
5. **Editor notes** — short bullets on what was cut, what was kept, and why.
6. **B-roll notes (optional)** — only if the script implies visual beats (texture close-up, board being passed, table reveal, seasonal setup).
