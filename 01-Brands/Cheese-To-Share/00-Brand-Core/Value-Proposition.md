## Core value proposition
Cheese To Share offers European-inspired foods that feel elevated, shareable, and memorable without making the buyer work hard to understand what to order or how to book.

## Everyday value
- Attractive food that feels different from standard fast-casual
- Menu categories that support sharing and occasion
- Easy order path through location/order partners

## Catering value
- Visually strong presentation
- Clear package entry points
- High social / hosting appeal
- Practical booking conditions
- Simple next-step conversion path

## Emotional value
- The food helps create a moment
- The buyer feels they chose something thoughtful
- The host feels the setup looks good enough to be remembered
- The brand feels warm, not generic

## Functional value
- Clear page architecture
- Direct order routing
- Straightforward inquiry flow
- Reliable contact path
- Packaging around actual use cases
