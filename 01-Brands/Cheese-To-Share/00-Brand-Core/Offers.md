## CTA logic
- **Order Now** = daily order intent → Location / ordering options
- **Book Now** = catering intent → inquiry / booking flow

## Offer architecture rule
CTS must keep **daily ordering** and **catering booking** separate. The menu page sells immediate food intent. The catering page sells event intent.

## High-value catering anchors
- Cheese Boards (easy packaged entry offer)
- Grazing Table (visual social experience)
- Floral Rustic Package (higher perceived value)
- Experience Stations (premium add-on lever)

## Operational note
Custom Styled Package should currently function as a **lead-generation / consultation path**, not a direct checkout path, because the service is not standardized enough for instant payment.
