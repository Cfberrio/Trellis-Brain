## Website KPIs
- Menu page → Location page click-through rate
- Location page → order platform click rate
- Catering page → Book Now click rate
- Catering form completion rate
- Contact form completion rate

## Sales / catering KPIs
- Lead-to-booking rate
- Inquiry-to-quote rate
- Quote-to-deposit rate
- Average catering ticket
- Add-on attach rate (stations / upgraded package)
- Repeat catering rate

## Retention KPIs
- Repeat buyer rate
- Repeat catering client rate
- Time to rebook
- Channel mix: Uber Eats / DoorDash / Pickup / catering

## Quality KPIs
- Response time to inquiries
- Deposit collection rate
- Balance collection before deadline
- Error-free event execution rate
