## Offer Ladder Summary
### Entry
Date-Lock Deposit. The first real conversion is the 50% deposit that holds the date.

### Core
Corporate Ready Room. Base venue rental with furniture, prep kitchen, bathrooms, and clear policies.

### Upsells
- Basic Package
- Presentation Pro / LED Package
- Workshop Broadcast Studio / Workshop Package
- Setup & breakdown
- Tablecloth rental
- Security when required
- Bar Service (see OEV-Beverage-Menu): House Beer & Wine ($18/guest), Essential Bar ($25.63/guest), Signature Bar ($32.13/guest), Bespoke Bar ($39.63/guest). Present as "Bar service, handled." — OEV coordinates, client selects package and pays at booking.

### Continuity
Corporate Calendar Membership / recurring room-time structure for repeat business.

### Downsells
- Feature downsell: remove production complexity
- Celebration Lite: compliant celebration path
- Personal Event Content Credit: barter-based promo path

## Offer Logic
Do not scatter these as random offers. They should behave like a ladder:
1. qualify fit,
2. secure date with deposit,
3. choose core rental path,
4. add production or execution vehicles only if the event requires them,
5. save no's with structured downsells instead of ad hoc discounting.
