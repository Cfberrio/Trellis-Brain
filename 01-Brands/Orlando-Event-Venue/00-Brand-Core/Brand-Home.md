# Orlando Event Venue - Brand Home

## Purpose
This note is the root operating hub for Orlando Event Venue inside the Trellis Brain. It is the first note that should be loaded for any OEV task that affects positioning, pricing, website structure, sales process, operations, or DNA.

OEV is positioned first as a professional, logistics-safe venue for corporate gatherings and workshops up to 90 guests, with celebration demand handled as a controlled secondary path rather than the main identity. ## Brand Definition
Orlando Event Venue is a local Orlando venue brand serving date-bound events where the buyer values certainty, compliance clarity, and smooth execution over the cheapest possible room. The strongest current fit is corporate planners and workshop hosts; celebration demand is real but should stay in a controlled schedule-filler lane rather than become the lead identity.

## What OEV Actually Sells
OEV does not just sell room access. It sells a lower-friction event outcome.

The practical offer stack is:
- compliant furnished venue access,
- a simple pricing entry point,
- optional production packages for AV / LED / streaming,
- optional execution add-ons for convenience,
- a deposit-first booking path that turns interest into commitment quickly.

## Primary Commercial Logic
The base economics only work when the lead can afford the true minimum booking threshold, accepts the operational rules, and has a real event date that creates urgency. That means OEV should not chase low-intent room shoppers; it should qualify fast and move good-fit leads into quote clarity, deposit collection, and add-on selection.

## What Must Stay True Across OEV
- OEV should sound clear, trustworthy, operational, and direct.
- OEV should not sound like a wedding fantasy brand or a generic banquet hall.
- OEV should not lead with "cheap" language.
- OEV should reduce friction by making rules, pricing floors, next steps, and operational requirements obvious.
- OEV should treat speed-to-clarity as part of the product, not just part of the sales process.
